# hls-react-player — Vite demo

Loads **`hls-react-player`** from the parent folder via `"file": ".."` in `package.json`. After the library is on npm, switch the dependency to a semver range (for example `^0.1.0`).

## Setup

From the package root (`hls-react-player/`):

```bash
npm run demo:install
npm run demo:dev
```

Or from `demo/`:

```bash
npm install
npm run dev
```

## Build

```bash
npm run demo:build
```

Output goes to `demo/dist/`.
